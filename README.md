# Face Emotion Detection AI System

## Project Description
A real-time face emotion detection system using Deep Learning and Computer Vision. Achieved **89.82% accuracy** on test data.

## Features
- Real-time face detection using OpenCV
- Deep Learning model with 89.82% accuracy
- Emotion classification: Angry, Happy, Sad, Surprise
- Live webcam integration with confidence scores

## Technology Stack
- Python
- TensorFlow/Keras
- OpenCV
- Convolutional Neural Networks (CNN)

## Model Performance
- **Final Test Accuracy:** 89.82%
- **Model Architecture:** Custom CNN with Batch Normalization
- **Training Data:** CIFAR-10 dataset (adapted)
